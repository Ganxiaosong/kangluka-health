# 康录卡——跨院慢病病史整合的轻量健康档案网页

## 项目简介
专为慢病老人设计的轻量健康档案网页，整合跨院病史、用药记录，支持离线二维码快速展示就医信息。

## 线上演示
https://ganxiaosong.github.io/kangluka-health/
