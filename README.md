## 📌 项目简介

**健康智析**是一款面向慢性病患者及其家属的轻量级个人健康档案管理网页应用。

我国慢性病患者超过 **3 亿人**，一个患者平均在 **2.3 家医院**就诊，病历分散在各医院系统中互不相通。患者复诊时说不清既往病史、记不清用药情况，医生也难以快速了解患者完整健康档案，导致重复检查、诊疗效率低下。

**健康智析**正是为解决这一真实痛点而生：患者自主管理健康档案，跨院就诊记录一条时间轴全掌握，上传化验单 AI 自动解读，医生扫码即可查看完整病史。

## 🚀 功能特性

### 1. 个人健康档案
- 记录姓名、性别、年龄、血型、身高、体重、过敏史、既往病史
- 信息结构化卡片展示，一目了然
- 支持随时编辑更新

### 2. 跨院就诊记录（时间轴）
- 以时间轴形式展示历次就诊记录
- 支持门诊、住院、体检三种类型分类
- 记录医院、科室、诊断结果、用药情况
- 可上传多张检查报告图片，点击放大查看
- 数据云端存储，换手机登录账号不丢失

### 3. AI 智能分析化验单（核心亮点）
- **图片识别模式**：上传化验单/检查报告，OCR 自动识别文字，AI 大模型自动分析
- **手动输入模式**：支持输入血压、血糖、血脂、肝功、肾功、血常规等 14 项常见指标
- **AI 输出内容**：
  - 指标总览卡片（正常 / 偏高 / 偏低 一目了然）
  - 异常指标深度建议：🍽️ 饮食建议 + 🏃 运动建议 + 💤 生活调理
  - 综合健康方案（多指标组合风险提醒）
- 接入 **硅基流动 Qwen2.5-7B** 大模型，国内网络直接访问

### 4. 医生扫码查看
- 首页生成专属二维码
- 医生扫码即可查看患者完整档案（无需注册、无需登录）
- 查看模式隐藏编辑按钮，防止误操作
- 解决「患者说不清病史、医生问不到关键信息」的沟通难题

### 5. 吃药提醒
- 设置药品名称、提醒时间、剂量
- 到点浏览器自动弹出通知（需授权通知权限）
- 即使网页最小化也能提醒

### 6. PWA 渐进式 Web 应用
- 支持添加到手机主屏幕，像原生 App 一样使用
- 支持离线查看已加载数据
- 响应式设计，完美适配手机、平板、电脑

## 🛠️ 技术栈

| 层级 | 技术 | 说明 |
|------|------|------|
| 前端 | HTML5 + CSS3 + JavaScript | 原生开发，无框架，轻量快速 |
| 样式 | Flex/Grid 布局、CSS 动画、毛玻璃效果 | 现代化 UI 设计 |
| 数据库 | Supabase (PostgreSQL) | 云端数据库 + 身份认证 + 文件存储 |
| OCR | Tesseract.js | 前端本地识别化验单文字，不上传图片 |
| AI | 硅基流动 SiliconFlow (Qwen2.5-7B) | 国内可访问的免费大模型 API |
| 部署 | GitHub Pages | 免费托管，全球可访问 |
| PWA | Service Worker + Manifest | 支持离线访问和添加到主屏幕 |

## 📱 使用方式

1. 打开线上链接（建议使用 Chrome / Edge / Safari 浏览器）
2. 注册账号（邮箱 + 密码）
3. 完善个人健康档案
4. 添加就诊记录，上传检查报告图片
5. 进入「AI 分析」页，上传化验单或手动输入指标，获取 AI 健康建议
6. 设置吃药提醒
7. 点击「添加到主屏幕」，像 App 一样使用
